Admin Panel

Manage challenges, invite team members, and review scores.

Send invite email

The invitee will receive a magic-link email. On first click they'll be prompted to pick a handle and optional password. Admin invites only work when deployed to Vercel (the /api/invite serverless function requires the service role key).

Import Challenges
